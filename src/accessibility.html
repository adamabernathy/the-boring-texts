Accessibility Statement

Our Commitment

We are committed to making The Boring Texts accessible to everyone, including people with disabilities. We strive to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards.

Accessibility Features

  • Semantic HTML: The site uses proper heading hierarchy, landmark regions, form labels, and ARIA attributes for screen reader compatibility.
  • Keyboard Navigation: All interactive elements — forms, buttons, and links — are fully operable via keyboard.
  • Skip Navigation: A "Skip to main content" link is provided for keyboard users to bypass repeated navigation.
  • Focus Indicators: Visible focus outlines are displayed on all interactive elements to assist keyboard navigation.
  • Reduced Motion: Animations are disabled for users who have enabled the prefers-reduced-motion setting in their operating system.
  • Live Regions: Status messages (form validation, download confirmations) use ARIA live regions so screen readers announce them automatically.
  • Responsive Design: The layout adapts to different screen sizes and zoom levels up to 200%.
  • Color Contrast: Text and interactive elements meet WCAG AA contrast ratio requirements.

Known Limitations

  • The File System Access API ("Save As" dialog) is not supported in all browsers. A standard download fallback is provided for unsupported browsers.

Feedback

If you encounter any accessibility barriers while using this tool, please let us know by opening an issue on the GitHub repository. We take accessibility feedback seriously and will work to address reported issues.
